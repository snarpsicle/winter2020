LIDAR

This scene was created from lidar data. It was made to showcase power line points that have been extracted from an LAS file. The buildings, their rooftops, and the trees are to scale relative to the LAS point cloud. Esri offers an add-in package called local governments 3D basemaps. Here is a link to the download add-in package. The buildings were created by imputing a buildings footprint, a DTM, a DSM, a feature height, and a LAS dataset. The trees were created by imputing the LAS dataset along with class codes from that dataset that were vegetation, building footprints, and a DTM. Buildings can be created quickly with rule based styles to make each unique and trees can be made to look more realistic but sadly these styles do not transfer to AGOL, which is what you see here displayed. To download this project and explore the methods in ArcPro click here. Inside this project you will find a working model for extracting power lines. All you need is an LAS file and a power lines vector and it will do the rest. Links in the upper right hand corner of this website will guide you visually through the steps to extract these power line points. This web app is responsive and moblie friendly, try resizing the browser and watch the app change to mobile mode. Feel free to explore the power lines. Zoom, pan, and rotate this 3D world.
