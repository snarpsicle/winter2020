Primary Refine

After investigating the LAS file it was apparent that the power lines fell into a predefined class of high vegetation and first of many hits. This class in itself could be divided into 3 sub classes; trees, building edges, and power lines. My analysis started with creating a DTM, DSM, intensity, and feature height. The DTM used the average elevation of cells in a sampling value of 6 inches of the LAS dataset ground. The DSM used the maximum elevation of cells in a sampling value of 6 inches of the dataset high vegetation. The intensity used the average intensity of cells in a sampling value of 6 inches of the dataset high vegetation. To create the feature height only using high vegetation and not creating unwanted pyramids evolved the tools; raster calculator, reclassify, and con. The focal statistics tool then ran the feature height with the statistic type set to standard deviation. The focal statistic and the intensity were both reclassified to extract as best it could only power lines. This map shows both layers reclassified and a third layer called 1st power line pixel refine, which is where the focal statistic and the intensity overlap. Click the eye in the layer list to toggle on/off layers to better understand the primary refine. For the next steps in my analysis check out secondary refine.
